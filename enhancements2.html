Enhancements

For this assignment, I did make some enhancements beyond the basic Javascripts in the Tutorials. Here is a list of them:

  • Quiz Countdown Timer: This timer is for the quiz. When time is out, the form will be automatically submitted. If users start the timer after answering all questions, the form will also be submitted rightaway. For this, I make use of "setInterval" function and, in addition, conditional operator to show the time in form of 2-digit number. I learn to create the timer here:
    Source: https://www.youtube.com/watch?v=MLtAMg9_Svw

  • Congratulatory Firework The firework can only be triggered if user get the maximum score within allowed attempts. For the firework, I make use of canvas and JavaScript classes "Particle" and "Firework". In addition, I also used DOM addEventListener method to make the canvas's size changes automatically to fit the page, and enable user can create more firework more quickly by clicking on canvas. I learn to create the firework here:
    Source: https://www.youtube.com/watch?v=rutUcHyPFsA&t=287s
